End-to-end recruitment process
  • Job description
  • Placing job ads
  • Interviewing
  • Short-listing suitable candidates
  • Reference checking

Labour hire administration
  • Entering information into CRM
  • Determining shifts for an on-hire labour force
  • Communicating shifts to candidates
  • Time sheet administration
  • Using a well-established online CRM or adapting existing CRM

  • Processing time sheets

  • Processing invoices
